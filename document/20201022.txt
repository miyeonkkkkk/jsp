사용자 정보 수정 : UPDATE
사용자 정보 수정 화면 요청(memberUpdate.jsp) : /memberUpdate?userid=brown
사용자 정보 수정 화면에서는 테이블에 저장된 사용자의 정보가 input tag에 설정이 되어야 한다.
(단, 사용자 아이디는 수정의 대상이 아님, 사용자 수정을 못하게 막아야 한다.)

사용자 사진은 현재 등록된 사진을 이미지로 표현
그 밑에 만약 사용자가 새로운 사진으로 바꾸고 싶을 때 이미지를 업로드할 수 있는 input tag를 준비

이름, 별명, 주소, 비밀번호를 수정 할 수 있다.

수정 버튼 클릭시 사용자가 입력한 정보로 table에 update 할 수 있다.

** 만약에 사용자가 새로운 사진 파일을 업로드 하지 않았을 경우 기존 이미지를 유지

정상적으로 수정될 경우 해당 사용자 정보 상세 조회 페이지로 이동
